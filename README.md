# GitHubTutorial

This is a tutorial how to use GitHub.

```bash

git add README.md
git status
git rm --cached README.md
git status
git add README.md

git commit -m "first commit"
git branch -M master
git remote add origin https://github.com/JohnsonSii/GitHubTutorial.git
git push -u origin master

git add .
git commit -a
git push -u origin master

```
